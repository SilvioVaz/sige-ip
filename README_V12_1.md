# SIGE-IP V12.1 — Conclusão rápida

Alteração pontual sobre a V12, sem remoção ou reorganização de módulos.

## Inclusão
- Botão **✓ Concluir** em todo cadastro que possui campo Status.
- Ao concluir: status muda para Concluído (ou Finalizado para contratos), progresso vai para 100% quando aplicável e a data da última atualização é renovada quando existir.
- A conclusão fica registrada automaticamente na Auditoria completa.
- Itens já concluídos, finalizados ou cancelados não exibem o botão.
