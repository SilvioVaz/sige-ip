SIGE IP — VERSÃO LOCAL SEGURA PARA PC

COMO ABRIR
1. Mantenha toda esta pasta junta.
2. Abra index.html preferencialmente no Google Chrome ou Microsoft Edge.
3. Não use modo anônimo/privado.

ARMAZENAMENTO
- Os dados principais ficam no IndexedDB do navegador, com capacidade muito superior ao localStorage.
- Na primeira abertura, o sistema tenta migrar automaticamente os dados das versões anteriores.
- O menu Configurações mostra espaço utilizado, limite estimado, persistência e último backup.

PROTEÇÃO RECOMENDADA
1. Abra Configurações.
2. Clique em "Proteger armazenamento local".
3. Clique em "Escolher arquivo para backup automático".
4. Selecione uma pasta sincronizada pelo OneDrive ou Google Drive.
5. Faça também um backup manual semanal.

CÓPIAS AUTOMÁTICAS
- O sistema cria cópias locais periódicas e guarda até 15 versões.
- Em Configurações > Ver cópias automáticas, é possível restaurar uma versão anterior.
- Antes de restaurar backup ou dados de exemplo, o sistema preserva uma cópia do estado atual.

LIMITAÇÕES IMPORTANTES
- Formatar o PC, apagar o perfil do navegador ou limpar os dados do site pode eliminar o banco local.
- O backup externo é a proteção contra perda do computador ou do navegador.
- Anexos grandes aumentam o banco e o arquivo de backup. Mantenha documentos críticos também em pasta externa organizada.
- Esta versão não sincroniza automaticamente com outros aparelhos.
