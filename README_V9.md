# SIGE IP v9 — Plataforma Web Completa

Inclui a base integral da v8.5 e acrescenta Supabase/PostgreSQL, autenticação, sincronização em tempo real, migração do IndexedDB, backups em nuvem e integração oficial com todas as abas do Google Sheets por Edge Function.

A aplicação está pronta para implantação, mas precisa das credenciais do seu projeto Supabase e de uma conta de serviço Google. Veja `docs/IMPLANTACAO_COMPLETA.md`.
