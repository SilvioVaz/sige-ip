SIGE IP — VERSÃO MULTIPLATAFORMA

FUNCIONA EM:
- PC: Google Chrome ou Microsoft Edge
- Android: Google Chrome
- iPhone/iPad: Safari

PUBLICAÇÃO:
Hospede toda esta pasta em um endereço HTTPS, como GitHub Pages ou Netlify.

INSTALAÇÃO NO ANDROID:
1. Abra o link HTTPS no Google Chrome.
2. Toque no menu de três pontos.
3. Toque em "Instalar aplicativo" ou "Adicionar à tela inicial".
4. Confirme SIGE IP.

INSTALAÇÃO NO IPHONE:
1. Abra no Safari.
2. Compartilhar > Adicionar à Tela de Início.

INSTALAÇÃO NO PC:
Abra no Chrome/Edge e use o ícone de instalação na barra de endereço.

DADOS:
A mesma chave de armazenamento das versões anteriores foi preservada, então os dados já existentes no mesmo navegador continuam disponíveis.
Os dados ficam locais em cada aparelho. Use "Exportar backup" e "Importar backup" para transferir dados entre PC e celular.
Para sincronização automática em tempo real, será necessário hospedar uma versão com banco de dados, servidor e login.
