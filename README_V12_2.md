# SIGE-IP V12.2 — Conclusão universal

- Mantém integralmente a versão V12.
- O botão **✓ Concluir** aparece em toda linha que também possui **Alterar** e **Excluir**, mesmo nos módulos sem campo Status visível.
- Ao concluir, o registro recebe status Concluído (ou Finalizado para contratos), data de conclusão e registro na Auditoria.
- Itens concluídos exibem o selo **✓ Concluído** no lugar do botão.
