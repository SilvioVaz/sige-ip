# SIGE-IP v10 — Governança Organizacional

Esta versão preserva os módulos, dados locais, backups, sincronização e estrutura da versão v9 e acrescenta:

- Centro de Governança Organizacional com KPIs, mapa de calor, prioridades e integração com OKRs.
- Funções críticas e matriz de risco organizacional.
- Plano de sucessão.
- Universidade corporativa e treinamentos.
- Gestão do conhecimento.
- Continuidade do negócio, contingência e recuperação.
- Gestão de ideias.
- Melhorias, Kaizen e PDCA.
- Auditorias e não conformidades.
- Gestão de fornecedores.
- Comunicação interna.
- Indicadores e KPIs.
- Matriz RACI integrada ao módulo existente de responsabilidades.

## Atualização segura

1. Na versão atual, acesse Configurações e gere um backup JSON.
2. Substitua todos os arquivos do site pelos arquivos desta pasta.
3. Atualize com Ctrl+Shift+R.
4. Os dados existentes são mantidos no mesmo IndexedDB. Os novos módulos são criados vazios automaticamente.
5. Importe o backup apenas se os registros anteriores não aparecerem.

## Observação

Os registros de exemplo do novo módulo aparecem apenas em instalações sem dados. Em bases já existentes, cadastre as funções críticas reais pelo Centro de Governança.
