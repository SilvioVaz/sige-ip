# SIGE-IP V10.1 — Central de Relatórios Completa

Esta atualização preserva a base de dados e os módulos da versão V10 e corrige a Central de Relatórios.

## Relatórios acrescentados

- Executivo Geral com indicadores de governança.
- Governança Consolidada.
- Saúde por Departamento.
- Funções Críticas.
- Matriz de Risco Organizacional.
- Mapa de Calor Organizacional.
- Plano de Sucessão.
- Treinamentos e Universidade Corporativa.
- Gestão do Conhecimento.
- Continuidade do Negócio.
- POPs e Procedimentos.
- Ideias, Melhorias, Kaizen e PDCA.
- Auditorias e Não Conformidades.
- Fornecedores.
- Comunicação Interna.
- Indicadores e KPIs.
- Matriz RACI.
- Todos os relatórios anteriores foram preservados.

## Novas funções

- Filtro por empresa.
- Cabeçalho com data e identificação do SIGE-IP.
- Exportação do relatório selecionado em CSV.
- Impressão e salvamento em PDF pelo navegador.
- Mensagem clara quando não há registros para o filtro.

## Atualização segura

1. Exporte um backup JSON na versão atual.
2. Substitua os arquivos do site pelos desta pasta.
3. Use Ctrl+Shift+R para forçar a atualização.
4. Os dados existentes permanecem na mesma estrutura IndexedDB.
