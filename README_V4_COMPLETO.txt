PRESERVAÇÃO DE DADOS
A versão mantém a mesma base IndexedDB e migra os registros existentes. Antes de substituir os arquivos, exporte um backup JSON.

ATUALIZAÇÃO NO GITHUB
Envie todos os arquivos e pastas para a raiz do repositório. Depois use Ctrl+Shift+R. Em aplicativo instalado, feche e abra novamente. Caso persista a versão anterior, remova o aplicativo e instale novamente.
